Instructions to execute

A) To execute Pallindrome program 
	1) Directly right click on Pallindrome.java file present in com.longestPallindrome package and run as a java application
	2) In the console enter the text that needs to be tested and click enter the output will be displayed
	
B) To execute Phone directory program 
	1) Right click on the Main.java file present in the com.phoneDirectory package and run as a java application
	2) Enter the keys you want to search for Eg: avi Then it will show you all the results with that search key
	3) Few of the contacts are hardcoded in Main.Java file
